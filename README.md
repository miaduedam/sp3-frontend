# SP3 Frontend

## Description
This project is the frontend part of my Semester Project 3 (SP3), developed as part of my Computer Science (Datamatiker) studies.
The application is built using React and communicates with a backend REST API.

## Technologies
* JavaScript
* React
* Vite
* HTML
* CSS
* REST API

## Features
* Component-based user interface
* Communication with backend API
* Dynamic data rendering
* Form handling and validation
* Navigation between views

## Setup & Installation 
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
3. Start the development server:
   ```bash
   npm run dev
4. Make sure the the backend API is running
